# 상세모달 상단표-우측필드 연결 (2026-05-31)

## 기준

공고관리 상세모달은 상단에 선택 공고 row를 가로 표로 보여주고, 우측 패널에서 같은 row의 컬럼을 입력한다.
상단 표와 우측 패널은 같은 컬럼명을 기준으로 연결한다.

## 동작

1. 상단 선택 row의 컬럼 값을 더블클릭하면 우측 패널에서 같은 컬럼이 있는 탭으로 이동한다.
2. 이동 후 해당 입력칸을 화면 중앙 근처로 스크롤하고 포커스한다.
3. 우측 패널 입력칸을 클릭하거나 포커스하면 상단 선택 row 표가 같은 컬럼 위치로 가로 스크롤된다.
4. 연결된 상단 컬럼과 우측 입력칸은 잠깐 파란색으로 강조한다.

## 컬럼 폭

상단 선택 row 표는 메인 표의 컬럼 폭 저장값을 따른다.

우선순위:

1. `main:human`
2. `main:preprocess`
3. `main:collect`
4. 기본 컬럼 폭

따라서 공고관리 표에서 사용자가 조정한 마지막 컬럼 폭이 상세모달 상단 표에도 반영된다.
`참가자격`처럼 긴 값이 있는 컬럼도 클릭 전 메인 표 폭과 같은 기준으로 유지한다.

## 주의

- 컬럼명이 우측 패널 필드 목록에 없으면 이동하지 않는다.
- 같은 컬럼명이 여러 탭에 있으면 탭 순서상 먼저 정의된 탭을 사용한다.
- 이 기능은 row 데이터 컬럼을 추가하지 않고 UI 상태로만 처리한다.
