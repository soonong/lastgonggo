# 검증 기준

## 기본 원칙

최종출력 전에 이상한 값, 누락값, 검토필요 상태를 확인한다.

## 필수 검증

| 검증 항목 | 오류 예시 | 처리 |
|---|---|---|
| 공고번호 | 공고번호 없음 | 오류 또는 검토필요 |
| 공고명 | 공고명 없음 | 오류 또는 검토필요 |
| 발주처명 | 발주처 없음 | 검토필요 |
| 날짜 | 날짜 형식 오류 | 오류 |
| 금액 | 음수 금액 | 오류 |
| 특수실적 금액/면적/점수 | `특수실적=공연장[3000M]_준공면적`인데 면적 숫자 없음 | 상세모달에서 빨간 오류 |
| 검토상태 | 검토필요인데 최종확정 출력 | 출력 전 경고 |

## 출력 가능 기준

기본적으로 `최종확정` 상태만 최종 엑셀 출력 대상으로 한다.

예외적으로 사용자가 선택하면 `검토필요`도 포함할 수 있지만, 엑셀에 상태가 표시되어야 한다.

예시:

| 출력 옵션 | 설명 |
|---|---|
| 검토완료만 출력 | `최종확정`만 포함 |
| 검토필요 포함 출력 | `검토필요` 포함, 상태 컬럼 표시 |
