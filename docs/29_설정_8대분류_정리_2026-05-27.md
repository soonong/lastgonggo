# 설정 8대분류 정리

## 목적

설정 화면의 검은 좌측 메뉴와 흰색 설정 탭을 운영 기준에 맞춰 8개 대분류 중심으로 단순화한다.

## 기준 파일

- `data/settings/설정탭구조.csv`
- `data/settings/삭제키워드.csv`
- `data/settings/공사명_괄호정리.csv`

## 8대분류

1. 대쉬보드
2. 공고수집
3. 분류규칙
4. 공고체크 키워드
5. 마스터데이터
6. 검수
7. 내설정
8. 변경로그

## 적용 내용

- 검은 좌측 메뉴는 `설정탭구조.csv`의 `대분류`를 순서대로 읽어 만든다.
- 검은 좌측 메뉴 아이콘은 `설정탭구조.csv`의 `아이콘` 컬럼을 읽어 표시한다.
- 흰색 설정 패널은 선택된 `대분류` 안의 `소분류`와 `라벨`만 보여준다.
- 화면에 없는 기존 설정 탭은 `설정탭구조.csv`에서 제외하여 노출하지 않는다.
- 삭제키워드는 `id, 컬럼명, 검색키워드, 처리방법, 메모` 구조로 관리한다.
- 공사명 대괄호정리는 `id, 컬럼명, 검색키워드, 변경 후, 처리방법` 구조로 관리한다.
- 대쉬보드는 우측 상단 검수용 배지를 제거하고, `오늘 상태`와 `최근 변경`을 표시한다.
- 설정 저장 시 `data/settings/변경로그.csv`에 일시, 구분, 대상, 내용, 사용자, 메모를 기록한다.

## 전처리 연결

- 삭제키워드:
  - 예: `컬럼명=공사명`, `검색키워드=[취]`, `처리방법=열삭제`
  - 전처리 시 해당 컬럼 값에 키워드가 포함되면 대상 row를 제외한다.
- 공사명 대괄호정리:
  - 예: `검색키워드=[종합심사낙찰제]`, `변경 후=[종심제]`
  - 전처리 시 해당 컬럼의 키워드를 변경 후 값으로 치환한다.
