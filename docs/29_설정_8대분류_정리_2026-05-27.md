# 설정 8대분류 정리

## 목적

설정 화면의 검은 좌측 메뉴와 흰색 설정 탭을 운영 기준에 맞춰 8개 대분류 중심으로 단순화한다.

## 기준 파일

- `data/settings/설정탭구조.csv`
- `data/settings/삭제키워드.csv`
- `data/settings/공사명_괄호정리.csv`

## 8대분류

1. 대쉬보드
2. 공고수집
3. 분류규칙
4. 공고체크 키워드
5. 마스터데이터
6. 검수
7. 내설정
8. 변경로그

## 적용 내용

- 검은 좌측 메뉴는 `설정탭구조.csv`의 `대분류`를 순서대로 읽어 만든다.
- 검은 좌측 메뉴 아이콘은 `설정탭구조.csv`의 `아이콘` 컬럼을 읽어 표시한다.
- 흰색 설정 패널은 선택된 `대분류` 안의 `소분류`와 `라벨`만 보여준다.
- 화면에 없는 기존 설정 탭은 `설정탭구조.csv`에서 제외하여 노출하지 않는다.
- 삭제키워드는 `id, 컬럼명, 검색키워드, 처리방법, 메모` 구조로 관리한다.
- 공사명 대괄호정리는 `id, 컬럼명, 검색키워드, 변경 후, 처리방법` 구조로 관리한다.
- 대쉬보드는 우측 상단 검수용 배지를 제거하고, `오늘 상태`와 `최근 변경`을 표시한다.
- 설정 저장 시 `data/settings/변경로그.csv`에 일시, 구분, 대상, 내용, 사용자, 메모를 기록한다.
- 모든 설정 탭에서 중간 회색 제목/설명 헤더를 제거해 표 영역을 넓힌다.
- 설정 상단은 검색, 다크모드, 닫기만 남긴다.
- 상단 breadcrumb, A1 로드 상태 배지, 중복 나가기 버튼은 제거한다.
- 다크모드 버튼은 설정 화면 전체에 실제 색상 테마를 적용한다.
- 공고수집 탭 순서는 `공고분류컬럼 설정`, `API 연결` 순서로 둔다.
- 기존 `api 컬럼형식` 명칭은 화면에서 `공고분류컬럼 설정`으로 쓴다.
- 기존 `서버공고 api` 명칭은 화면에서 `API 연결`로 쓴다.
- 표 상단 제목과 건수는 한 줄로 압축한다.
- 표 버튼명은 `CSV 다운` 대신 `내보내기`, `CSV 업로드` 대신 `가져오기`로 쓴다.
- 표에서 값을 바꾼 뒤 다른 설정 탭으로 이동하거나 설정을 닫으면 저장 여부를 먼저 확인한다.

## 공고분류컬럼 설정

공고분류컬럼 설정은 서버공고 컬럼, 파서맨 수집값, 계산맨 규칙을 최종 표준 컬럼으로 연결하는 기준표다.

| 컬럼 | 운영 의미 |
|---|---|
| 처리방법 | `계산`, `수집`, `둘다`, 공란 중 하나. 수집은 서버공고 또는 파서맨 수집값이고, 계산은 참조방법 코드로 값을 만든다. |
| 우선순위 | 같은 컬럼에 서버정보와 파서맨정보가 있을 때 어떤 출처를 쓸지 정한다. |
| 참조방법 | 처리방법이 `계산` 또는 `둘다`일 때 사용하는 코드형 규칙이다. |
| 참조메모 | 참조방법 코드가 무엇을 계산하거나 조회하는지 사람이 읽을 수 있게 남긴 설명이다. |
| 컬럼입력목록 | 기존 `선택목록`의 화면 표시명. 해당 컬럼에 입력 가능한 목록이다. |
| 공고관리 표시 | 체크박스로 편집한다. 공고관리 표에서 필요한 컬럼만 보이게 조정하는 기준이다. |
| 상세정보입력 | 체크박스로 편집한다. 상세정보 입력 화면에서 필요한 컬럼만 보이게 조정하는 기준이다. |

`서버공고일치`, `확정메모`는 과거 서버공고 비교 검증과 컬럼 확정 과정에서 쓰던 메타다. 현재 운영 입력에는 핵심 컬럼이 아니므로 기본 화면에서는 숨기고, 데이터 파일에는 유지한다.

## 설정 화면 정리 원칙

설정 화면을 수정할 때는 먼저 요구사항을 문서로 정리한 뒤 작업한다.

예:

1. 제거 대상과 연결된 JSX, 상태값, import, CSS를 검색한다.
2. 단순 숨김 처리 대신 사용하지 않는 코드까지 정리한다.
3. 빌드와 브라우저 검수 후 문서를 갱신한다.
4. 의미 있는 작업 단위로 git commit/push 한다.

## 전처리 연결

- 삭제키워드:
  - 예: `컬럼명=공사명`, `검색키워드=[취]`, `처리방법=열삭제`
  - 전처리 시 해당 컬럼 값에 키워드가 포함되면 대상 row를 제외한다.
- 공사명 대괄호정리:
  - 예: `검색키워드=[종합심사낙찰제]`, `변경 후=[종심제]`
  - 전처리 시 해당 컬럼의 키워드를 변경 후 값으로 치환한다.

  - `서버공고일치`, `확정메모`는 기본 화면에서 숨김.
