## 1차 MVP 구현 범위

웹 앱 뼈대를 만들고 수집 화면을 먼저 구현했다.

구현 위치:

```text
app/
  server.mjs
  src/App.tsx
  src/api.ts
  src/types.ts
  src/App.css
```

## 실행 포트

기존 프로세스가 `3001`, `5173`을 사용 중이어서 새 앱은 아래 포트를 쓴다.

```text
API 서버: http://127.0.0.1:3101
웹 앱:    http://127.0.0.1:5174
```

예시:

```text
3001에 기존 서버가 떠 있음
5173에 기존 Vite가 떠 있음
  -> 새 앱은 3101/5174로 실행
```

## CSV 처리 결정

CSV는 날짜와 숫자 문자열을 임의 변환하지 않고 원문 문자열로 보존한다.

예시:

```text
원본 입력일=2026-04-20 15:17:00
화면 표시=2026-04-20 15:17:00
```

`4/20/26`처럼 미국식 날짜로 바꾸지 않는다.
