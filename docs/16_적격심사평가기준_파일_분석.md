## 파일 성격

이 파일은 공고의 `원발주처`, 건설유형, 금액, 공고일을 기준으로
`적격심사기준`과 최종 `적격평가기준_세부`를 찾기 위한 매핑표로 본다.

기존 `적격심사기준룰.md`의 처리 순서 중 아래 단계에 해당한다.

```text
원발주처 + 주공종 건설유형 + 추정가격 + 공고일/입력일 기준으로 적격평가기준_세부 결정
```

## 확정된 룰

### 목적

이 파일의 모든 분류는 A4 `적격심사세부기준`을 정확히 찾기 위한 것이다.
적격심사평가기준을 잘못 고르면 만점배수, 평가기준금액, 종목세부JSON이 모두 틀어지므로
확실하지 않은 값은 자동 확정하지 않는다.

### 비교 금액

이 파일에서 금액 구간을 고를 때는 `추정가격`을 사용한다.

예시:

```text
추정가격=450,000,000
기초금액=510,000,000
원발주처=행자부
건설유형=일반건설
  -> 450,000,000으로 적격심사평가기준 행을 고른다.
```

### 금액 경계

금액 컬럼은 `미만` 기준이다.
기준 금액과 정확히 같으면 해당 행이 아니라 다음 금액 구간을 확인한다.

예시:

```text
적격심사기준=2억미만
일반건설 (원)=200000000
추정가격=199999999
  -> 적용

추정가격=200000000
  -> 미적용, 다음 구간 확인
```

### 시행일

`시행일`은 공고일 또는 입력일과 비교해 어느 기준을 가져와야 하는지 판단하기 위해 쓴다.
공고일 당시 시행 중인 기준만 후보가 된다.

같은 `원발주처` 안에 서로 다른 시행일이 섞일 수 있으므로 행 단위로 판단한다.

예시:

```text
원발주처=한국전력공사
공고일=2026-02-01
시행일=2026-01-31
  -> 후보 가능

공고일=2026-01-20
시행일=2026-01-31
  -> 후보 제외
```

### 적격평가기준_세부

`적격평가기준_세부`는 아래 형식으로 만든다.

```text
적격평가기준_세부 = 원발주처 + "_" + 적격심사기준
```

예시:

```text
원발주처=행자부
적격심사기준=별표-4
  -> 적격평가기준_세부=행자부_별표-4
```

```text
원발주처=한국전력공사(배전)
적격심사기준=10억미만
  -> 적격평가기준_세부=한국전력공사(배전)_10억미만
```

### 복합 종목

현재는 주공종 기준으로만 적격심사평가기준을 고른다.
종목별 적격심사기준 개별 호출은 향후 확장 후보로 남긴다.

예시:

```text
단독평가종목=(건축,전기)
주공종=건축
  -> 일반건설 기준으로 적격심사평가기준을 고른다.
```

### 원문 보존

`적격심사기준` 표기는 원문 그대로 보존한다.
A4 호출용 값이 필요하면 `적격평가기준_세부`에서 조합한다.

예시:

```text
한국도로공사 적격심사기준=별표5
행자부 적격심사기준=별표-5
한국전력공사 적격심사기준=50억미만
  -> 원문 그대로 보존
```

## 보류 사항

현재는 주공종 기준만 사용한다.
종목별 평가기준금액에 따른 개별 기준 호출은 나중에 별도로 검토한다.

예시:

```text
단독평가종목=(토목,전기)
토목 평가기준금액=800,000,000
전기 평가기준금액=200,000,000
  -> 현재는 주공종 기준만 사용
  -> 향후 종목별 개별 A4 호출 필요 여부 검토
```
