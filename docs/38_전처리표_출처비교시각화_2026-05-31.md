# 전처리표 출처 비교 시각화 (2026-05-31)

## 목적

입찰공고 전처리 표에서 각 셀의 최종값이 `서버공고`와 `문서곡괭이` 중 어떤 출처와 일치하는지 한눈에 보여준다.

작업자는 노랑 셀만 우선 확인하면 된다.

## 출처 정의

| 출처 | 의미 |
|---|---|
| 서버공고 | A1 서버공고에서 내려온 값 또는 문서곡괭이 실행 직전 row의 값 |
| 문서곡괭이 | A3 파싱용공고문에서 문서곡괭이 룰로 추출한 값 |
| 최종값 | 컬럼 우선순위와 병합 규칙을 거쳐 표에 표시되는 값 |

주의: `row[col]`은 이미 문서곡괭이 값으로 덮였을 수 있으므로 서버공고 원본값으로 보지 않는다.

## 저장 구조

비교용 메타는 숨김 컬럼 `출처비교JSON`에 JSON 문자열로 저장한다.

예시:

```json
{
  "추정가격": {
    "서버공고": "1200000000",
    "문서곡괭이": "1200000000",
    "선택출처": "문서곡괭이",
    "판정": "일치",
    "조건판단형태": "1_2",
    "rule_id": "6",
    "키워드": "추정가격",
    "근거": "추정가격: 금 1,200,000,000원"
  }
}
```

`출처비교JSON`은 화면 컬럼에는 보이지 않는다.

## 색상 규칙

| 서버공고 | 문서곡괭이 | 판정 | 색 |
|---|---|---|---|
| 있음 | 있음 + 같음 | 일치 | 녹색 |
| 있음 | 있음 + 다름 | 불일치 | 노랑 |
| 있음 | 없음 | 서버공고만 | 노랑 |
| 없음 | 있음 | 문서곡괭이만 | 노랑 |
| 없음 | 없음 | 저장 안 함 | 무색 |

현재는 문서곡괭이 결과에 들어온 컬럼을 기준으로 비교 메타를 만든다.
문서곡괭이가 시도했지만 빈값인 컬럼까지 표시하려면 문서곡괭이 결과가 빈 후보 컬럼도 반환해야 한다.

## 툴팁

전처리 표에서 비교 메타가 있는 셀에 마우스를 올리면 다음 정보를 보여준다.

- 최종값
- 선택출처
- 서버공고 값
- 문서곡괭이 값
- 판정
- 조건판단형태, rule id, 키워드
- 근거 본문
