# 상세모달 첨부파일 미리보기 점검 메모 (2026-05-30)

## 변경 기준

- 공고관리 상세모달에서 첨부파일을 클릭하면 좌측 공고문 영역에 새 탭으로 연다.
- PDF, 이미지, HTML, TXT는 로컬 서버의 `/api/file-preview` 프록시를 거쳐 inline 미리보기로 보여준다.
- HWP는 로컬 서버의 `/api/hwp-preview`에서 `pyhwp(hwp5)` 변환기로 HTML 변환 후 좌측 탭에 바로 보여준다.
- HWP 변환 시 `index.xhtml`만 쓰지 않고 `styles.css`도 함께 생성해 HTML 안에 주입한다. 이 CSS를 빼면 한글 자간, 문단, 표 서식이 원문보다 크게 깨진다.
- HWPX는 아직 로컬 변환 대상이 아니며, 변환 실패 안내와 원본 열기를 제공한다.
- 원본 파일은 항상 `원본 열기` 링크로 열 수 있게 둔다.

## 남은 결정

- HWP 변환기는 사용자 PC의 Python `hwp5` 패키지를 사용한다. 변환 실패 시 해당 파일은 원본 열기로 확인한다.
- XLS/XLSX, ZIP은 현재 미리보기 대상이 아니라 원본 열기/다운로드 대상이다.

## 상세모달 UI 정리

- 우측 패널의 `지역 매핑 검토필요` 카드는 제거한다.
- 지역 검토가 필요한 필드는 기존처럼 입력칸 강조만 유지한다.
- 하단 버튼은 `저장 / 닫기` 하나만 둔다.
- 상단 선택 공고 row 영역은 한 줄만 보이게 높이를 줄이고, 가로 스크롤은 유지한다.
