# 종목묶기 케이스 매트릭스

## 목적

이 문서는 사용자가 던져준 종목 예시를 기준으로 모든 종목묶기 케이스를 명확히 만드는 작업장이다.
질문부터 던지지 않고, 먼저 케이스를 만든 뒤 확정/추정/확인필요 상태를 표시한다.

```text
사용자 예시
  -> 케이스 ID 부여
  -> 조건 분해
  -> 종목 결과
  -> 단독평가종목 결과
  -> 상태 표시
  -> 구현 테스트 케이스로 전환
```

## 상태값

| 상태 | 의미 | 처리 |
|---|---|---|
| 확정 | 사용자가 직접 예시로 준 결과 | 구현 기준으로 사용 가능 |
| 예시기반 추정 | 사용자가 준 예시에서 일반화한 결과 | 테스트 전 확인 필요 |
| 확인필요 | 예시 사이에 충돌이 있거나 아직 기준이 비어 있음 | 임의 구현 금지 |

## 케이스 축

종목묶기 케이스는 아래 축의 조합으로 만든다.

| 축 | 값 | 예시 |
|---|---|---|
| 상호진출여부 | `0`, `1` | `상호진출여부=0`, `상호진출여부=1` |
| 주공종 건설유형 | 일반, 기타, 전문 | `건축/전기`는 일반 주공종 |
| 포함 건설유형 | 일반, 기타, 전문, 복합 | `토목/전기/토공`은 일반+기타+전문 |
| 전문건설 주력종목 | 없음, 1개, 여러 개 | `철콘`, `지반조성포장_토공` |
| 상위종목 | 없음, 토건, 전문소방 | `건축 -> 토건`, `기계소방 -> 전문소방` |
| 대체종목 | 없음, 있음 | `기계가스설비 -> 가스난방` |
| 상호진출 상대종목 | 없음, 있음 | `실내건축 -> 건축`, `지반조성포장 -> 건축/토목` |
| 종목모두보유 | `0`, `1` | `1`이면 모두보유 묶음, `0`이면 개별 옵션 |
| 공동도급 조건 | 공동이행, 분담이행, 주계약자관리 | 종목모두보유와 같은 포지션으로 처리 |
| 입력순서 | 보존 | `전기/건축`과 `건축/전기`는 다른 순서로 표시 |

## 상호진출=1: 종합공사

## 구현용 일반화 규칙 초안

### 상호진출=0 일반 규칙

```text
1. 입력 순서의 첫 종목을 주공종으로 본다.
2. 전문건설은 표시용 종목에서는 대업종으로 합친다.
3. 전문건설은 단독평가종목에서는 대업종_주력종목으로 분해한다.
4. 주력종목이 없으면 대업종 자체를 단독평가종목으로 쓴다.
5. 상위종목이 있으면 표시용 종목에 대체 옵션을 추가한다.
6. 종목모두보유=1, 공동이행, 분담이행, 주계약자관리는 모두보유 묶음으로 처리한다.
7. 종목모두보유=0이면 하위 종목과 상위종목을 각각 개별 옵션으로 둔다.
8. 기타건설은 상호진출 대상이 아니므로 옵션마다 그대로 동반한다.
```

예시:

```text
토목/토공/포장
  -> 표시용 전문건설: 지반조성포장
  -> 심사용 전문건설: 지반조성포장_토공, 지반조성포장_포장
  -> 종목: (토목,지반조성포장)/(토건,지반조성포장)
```

### 상호진출=1 종합공사 규칙

```text
1. 일반 주공종을 기준으로 일반 옵션, 상위종목 옵션, 전문 참여 옵션을 만든다.
2. 전문 참여 옵션의 단독평가종목에는 상호진출 매핑표에 근거한 일반종목 접미사를 붙인다.
3. 매핑되는 일반종목이 둘 다 있으면 입력순서 기준 첫 매칭을 자동값으로 둔다.
4. `(건축,토목)`처럼 일반종목 묶음 전체를 접미사로 붙이는 케이스는 없다.
5. 기타건설이 있으면 각 옵션에 그대로 동반한다.
6. 입력순서를 보존한다.
```

예시:

```text
토목/전기/토공
  -> 종목: (토목,전기) / (토건,전기) / (지반조성포장,전기)
  -> 단독평가종목: (토목,전기) / (지반조성포장_토공_토목,전기)
```

### 상호진출=1 전문공사 규칙

```text
1. 전문 주공종 옵션, 일반 참여 옵션, 일반 상위종목 표시 옵션을 만든다.
2. 일반 참여 옵션의 단독평가종목에는 상호진출 매핑표에 근거한 전문종목 접미사를 붙인다.
3. 토건은 어디서든 단독평가종목에서 평가하지 않으므로 아예 넣지 않는다.
4. 매핑되는 전문종목이 둘 다 있으면 입력순서 기준 첫 매칭을 자동값으로 둔다.
5. 기타건설이 있으면 각 옵션에 그대로 동반한다.
6. 입력순서를 보존한다.
```

예시:

```text
토공/전기/토목
  -> 종목: (지반조성포장,전기) / (토목,전기) / (토건,전기)
  -> 단독평가종목: (지반조성포장_토공,전기) / (토목_지반조성포장_토공,전기)
```

## 종목세부JSON 전개 규칙

종목세부JSON은 `단독평가종목`의 구분자 의미를 보존하지 않는다.
슬래시, 괄호, 쉼표를 해석한 뒤 앞에서부터 순서대로 모든 종목을 넣는다.
단, 전개 전에 단독평가종목 문자열 자체가 잘못되어 완전히 같은 옵션이 반복되는 경우에는 알림을 띄운다.
일부 종목이 겹치더라도 옵션 전체가 다르면 정상으로 보고 순서대로 넣는다.

| ID | 단독평가종목 | 종목세부JSON 입력 순서 | 상태 | 메모 |
|---|---|---|---|---|
| JM-JSON-001 | `a/(b,c)/d` | `a`, `b`, `c`, `d` | 확정 | 구분자 정보 없이 순서대로 전개 |
| JM-JSON-002 | `지반조성포장_포장/지반조성포장_토공` | `지반조성포장_포장`, `지반조성포장_토공` | 확정 | 옵션그룹 저장 없음 |
| JM-JSON-003 | `(지반조성포장_포장,지반조성포장_토공)` | `지반조성포장_포장`, `지반조성포장_토공` | 확정 | 필수그룹 저장 없음 |
| JM-JSON-004 | `(건축,전기)` | `건축`, `전기` | 확정 | 표시용 종목이 `(건축,전기)/(토건,전기)`이어도 단독평가종목에는 토건 옵션을 넣지 않음 |
| JM-JSON-005 | `전기/전기` | 알림 | 확정 | `a/a`, `(a,b)/(a,b)`처럼 같은 옵션 반복은 꼬인 값으로 보고 알림 |
| JM-JSON-006 | `(a,b)/(a,c)` | `a`, `b`, `a`, `c` | 확정 | 겹치는 종목이 있어도 옵션이 다르므로 순서대로 전개 |

## 종목세부JSON 금액 케이스

종목세부JSON의 금액 기준은 `평가기준금액`이다.
평가기준금액은 적격심사세부기준의 배수적용기준에 따라 추정가격, 기초금액, 종목별 금액 등으로 달라질 수 있다.
비율은 기본 계산하지 않고 금액을 기준으로 한다.
공고문에 금액은 없고 비율만 있을 때는 역산할 수 있다.

| ID | 조건 | 입력 예시 | 종목세부JSON 금액 | 상태 | 메모 |
|---|---|---|---|---|---|
| JM-AMT-001 | 슬래시 옵션 | `지반조성포장_포장/지반조성포장_토공`, 평가기준금액=1억 | 둘 다 1억 | 확정 | 각각 참여 가능하므로 각각 평가금액 필요 |
| JM-AMT-002 | 괄호 묶음, 종목별 금액 있음 | `(토목,전기)`, 토목=8억, 전기=2억 | 토목 8억, 전기 2억 | 확정 | 비율 계산은 하지 않음 |
| JM-AMT-003 | 괄호 묶음, 종목별 금액 없음 | `(토목,전기)`, 평가기준금액=10억 | 토목 10억, 전기 빈값 | 확정 | 없으면 첫 번째에 전체 평가기준금액을 넣고 나머지는 빈값으로 두어 사람이 입력 |
| JM-AMT-004 | 괄호 묶음, 비율만 있음 | `(토목,전기)`, 토목 80%, 전기 20%, 평가기준금액=10억 | 토목 8억, 전기 2억 또는 빈값 | 확정 | 확실히 역산 가능하면 금액만 채우고, 불확실하면 빈값. 별도 산출근거 메모는 자동 생성하지 않음 |
| JM-AMT-005 | 대체종목 | 종목=`기계가스설비/가스난방`, 단독평가종목=`기계가스설비_기계설비` | `기계가스설비_기계설비`만 생성 | 확정 | 대체종목은 종목/단독평가종목 생성 단계에서 처리 |
| JM-AMT-006 | 토건 표시 옵션 | 종목=`건축/토건`, 단독평가종목=`건축` | `건축`만 생성 | 확정 | 토건은 단독평가종목에 없으므로 JSON에도 없음 |
| JM-AMT-007 | 전문소방 | 단독평가종목=`(전기,전문소방)` | 전기, 전문소방 생성 | 확정 | 토건만 제외, 전문소방은 포함 |

## 종목만점실적 케이스

종목만점실적은 `평가기준금액 x 만점배수`로 계산한다.
종목별 평가기준금액이 있으면 각 종목의 금액을 기준으로 계산한다.

| ID | 조건 | 입력 예시 | 종목만점실적 | 상태 | 메모 |
|---|---|---|---|---|---|
| JM-FULL-001 | 단일 기준금액 | 평가기준금액=5억, 만점배수=1.0 | 5억 | 확정 | 평가기준금액은 추정가격/기초금액 등 기준에 따라 결정 |
| JM-FULL-002 | 종목별 금액 있음 | 토목=8억, 전기=2억, 만점배수=1.0 | 토목 8억, 전기 2억 | 확정 | 종목별 평가기준금액 기준 |
| JM-FULL-003 | 종목별 적격심사기준 필요 가능성 | 토목/전기 복합 공고 | 메모 | 보류 | 현재는 주공종 기준으로 A4를 호출하고, 종목별 개별 호출은 향후 확장 후보 |
