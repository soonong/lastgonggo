# 출력 룰

## 기본 원칙

최종 엑셀은 검증 기준을 통과한 데이터를 출력한다.
출력 컬럼은 일부 대표 컬럼만 고르지 않고 모든 표준 컬럼을 포함한다.
컬럼 순서와 양식은 기존 사용 중인 최종 엑셀 양식에 가급적 맞춘다.

예시:

```text
공고번호, 공고명, 발주처명, 종목, 지역제한, 기초금액만 출력하는 방식이 아니다.
서버공고와 전처리에서 확정한 전체 표준 컬럼을 출력한다.
```

## 출력 옵션

| 옵션 | 설명 | 예시 |
|---|---|---|
| 검토완료만 출력 | `최종확정` 상태만 출력 | 납품용 최종 파일 |
| 검토필요 포함 출력 | `검토필요`도 출력하되 상태 컬럼 표시 | 내부 검토용 파일 |
| 발주처별 출력 | 서버공고의 발주처 컬럼 기준으로 필터 출력 | 한국전력공사 공고만 출력 |
| 날짜별 출력 | 특정 수집일 또는 공고일 기준 출력 | 2026-05-22 공고만 출력 |

## 파일명 규칙

```text
YYYY-MM-DD_입찰공고_최종분류.xlsx
```

예시:

```text
2026-05-22_입찰공고_최종분류.xlsx
```
