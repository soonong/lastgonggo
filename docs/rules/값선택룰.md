# 값 선택 룰

## 기본 우선순위

```text
사람 입력값 > 코드 규칙값 > 공고문에서 명확히 추출한 값 > AI 후보값
```

## 선택 기준

| 후보 | 사용 기준 | 예시 |
|---|---|---|
| 사람 입력값 | 있으면 최우선 사용 | 사람이 `지역제한=서울`로 검증 |
| 코드 규칙값 | 명확한 룰로 확정 가능할 때 사용 | `서울교육청` -> `서울특별시교육청` |
| 공고문 추출값 | 문구가 명확할 때 후보 또는 확정 | `서울특별시 소재 업체` -> `서울` |
| AI 후보값 | 근거가 있고 확실할 때만 사용 | 공고명과 공고문이 모두 전기공사로 명확 |

## 충돌 처리

후보값이 서로 다르면 자동 확정하지 않고 검토 대상으로 남긴다.

예시:

```text
API 업종: 기타
공고문 추출 업종: 전기공사업
처리: 사람입력 화면에서 충돌값으로 표시한다.
```
