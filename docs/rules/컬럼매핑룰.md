# 컬럼 매핑 룰

## 기본 원칙

발주처마다 다른 원본 컬럼명을 표준 컬럼으로 매핑한다.

## 예시

| 발주처 | 업무유형 | 원본 컬럼 | 표준 컬럼 | 이유 |
|---|---|---|---|---|
| 나라장터 | 공통 | `bidNtceNm` | 공고명 | 나라장터 공고명 필드 |
| 나라장터 | 공통 | `bidNtceNo` | 공고번호 | 나라장터 공고번호 필드 |
| 나라장터 | 공통 | `dminsttNm` | 발주처명 | 수요기관명 |

## 변경 감지

원본에 새 컬럼이 생기거나 기존 컬럼이 사라지면 검토 대상으로 표시한다.

예시:

```text
기존에는 `bidNtceNm`이 있었는데 새 데이터에서 `noticeName`만 들어오면
자동 매핑하지 않고 `검토필요`로 표시한다.
```
