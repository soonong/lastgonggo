# 나라장터 시설 업종 기타 예시

## 상황

나라장터 시설 공고에서 API 업종은 `기타`로 들어왔지만, 첨부 공고문에는 실제 업종이 명확하게 적혀 있다.

## 원본값

| 항목 | 값 |
|---|---|
| 발주처 | 나라장터 |
| 업무유형 | 시설 |
| API 업종 | 기타 |

## 후보값

| 출처 | 후보값 | 근거 |
|---|---|---|
| API | 기타 | 원본 수집값 |
| 공고문 파싱 | 전기공사업 | 공고문에 `전기공사업 등록업체` 문구 존재 |

## 처리

API 값과 공고문 파싱값이 충돌하므로 자동 확정하지 않고 사람입력 화면에서 검토 대상으로 표시한다.

## 최종 결과 예시

사람이 `전기공사업`으로 검증하면 최종 업종은 `전기공사업`이 된다.

재전처리 대상:

```text
종목그룹
공고분류
키워드
```
