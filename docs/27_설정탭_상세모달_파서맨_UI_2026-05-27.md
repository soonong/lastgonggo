## 구현한 내용

1. 설정 화면의 좌측 메뉴를 `data/settings/설정탭구조.csv` 기준 31개 탭으로 세분화했다.
2. 설정 탭 구조는 코드 상수 배열이 아니라 CSV에서 읽는다.
3. 종목매핑 화면에 `상호진출_상대종목 관리` 편집 패널을 추가했다.
4. 종목매핑 전체 표는 기존 공통 테이블 기능을 그대로 사용한다.
5. 지역 매핑 후보/검토필요는 상세모달 우측 상단 경고 배너와 해당 필드 강조로 표시한다.
6. 상세모달 좌측에 공고문 검색 입력과 하이라이트 뷰를 추가했다.
7. 상세모달 첨부 탭에서 A2 첨부파일 목록을 불러오고, 선택한 첨부를 좌측 탭으로 연결한다.
8. 파서맨 룰 편집의 `조건판단형태` 입력을 드롭다운으로 바꾸고, 선택한 조건의 용도/예시/주의사항을 즉시 표시한다.
